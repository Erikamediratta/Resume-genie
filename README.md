# ✨ Resume Genie - AI-Powered Career Assistant

A multi-tool AI career assistant built with Google Gemini API and Streamlit.

🔗 **Live Demo:** [Click Here](https://resume-genie-jfpe4e3rem4wtqfmymnepd.streamlit.app/)

---

## 🛠️ Tools Included

| Tool | Description |
|------|-------------|
| 📋 Resume Checker | AI feedback on strengths, weaknesses and missing keywords |
| 🎯 Resume Scorer | Score your resume out of 100 with detailed breakdown |
| ✉️ Cover Letter Generator | Generate tailored cover letters from your resume + JD |
| 🎓 AI Career Coach | Chat with an AI coach using your resume as context |

---

## 🚀 Tech Stack

- Python
- Google Gemini API (gemini-2.5-flash)
- Streamlit
- pdfplumber
- python-dotenv

---

## ⚙️ Run Locally

1. Clone the repo
```bash
git clone https://github.com/Erikamediratta/Resume-genie.git
cd Resume-genie
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Create `.env` file
